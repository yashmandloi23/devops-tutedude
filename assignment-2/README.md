# Q1 → Grade Checker

## Problem Statement
Take marks as input from the user and print the grade based on:

90+  → A  
80-89 → B  
70-79 → C  
60-69 → D  
Below 60 → F  

## Concept Used
- if-elif-else statements
- User input
- Conditional logic

## File Name
grade_checker.py

## What the code does
- Takes score from user
- Checks conditions using if-else
- Prints corresponding grade
